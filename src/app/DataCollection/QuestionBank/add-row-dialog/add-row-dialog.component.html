Añadir Intent

Se conoce a Intent como lo que permite iniciar una actividad en una app mediante la descripción de una acción simple que se desea realizar. En este caso, el Intent definira las respuestas que dara el bot hacia los mensajes de los estudiantes, es decir que se puede ir entrenando las respuestas del bot con supuestos mensajes que el estudiante puede enviar
